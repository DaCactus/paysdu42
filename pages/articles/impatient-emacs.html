Impatient Emacs

Petite information, pour mieux comprendre ce qui va suivre: C- veut dire 'appuyez sur la touche control'. M- veut dire 'appuyez sur la touche Meta', dépend de votre configuration, la plupart du temps c'est la touche Alt, mais il arrive que ce soit la touche Esc par exemple.

Débuter avec Emacs

  • Ouvrir emacs : Taper emacs -nw dans votre terminal.
  • Mettre un header / affiche l'aide : C-c C-h
  • Sauvegarder : C-x C-s
  • Fermer emacs : C-x C-c
  • Ouvrir un fichier : emacs "chemin du fichier"
  • Ouvrir un fichier dans emacs déjà ouvert : C-x C-f
  • Changer le mode d'emacs (couleurs) : M-x 'mode'-mode (exemple : C-mode, html-mode, ect)
  • Annuler ce que je suis en train de demander : C-g
  • Annuler (revenir en arrière) : C-/
  • Ne pas utiliser en mode graphique : emacs -nw

Partager la fenêtre emacs

  • Séparer la fenêtre en 2 verticalement : C-x 3
  • Séparer la fenêtre en 2 horizontalement : C-x 2
  • Aller à l'autre fenêtre : C-x o
  • Supprimer une fenêtre : C-x 0

Fonctionnalités sympas

  • Faire une tabulation : M-i
  • Complèter automatiquement un mot déjà écrit : M-/
  • Rechercher un mot : C-s
  • Remplacer un mot par un autre : M-x 'replace-string'
  • GDB dans emacs : M-x 'gdb'. A ouvrir avec le partage de fenêtre pour voir les étapes une par une dans le fichier source.
  • Parcourir un dossier : M-x d
  • Sélectionner une région : C-espace puis les fleches haut et bas pour selectionner.
    Arreter la sélection avec C-g
  • Tout dépendra du mode dans lequel vous travaillez pour les commentaires. Le plus facile est 'M-x comment-region' pour commenter et 'M-x uncomment-region' pour dé-commenter.
  • Commenter la selection (c-mode): C-c C-c
  • Dé-commenter : C-u C-c C-c
  • Couper depuis le curseur jusqu'à la fin de la ligne : C-k
  • Couper une région : C-w
  • Coller depuis le curseur jusqu'à la fin de la ligne : C-y
  • Revenir au début de la ligne : C-a
  • Aller à la fin de la ligne : C-e
  • Inverser deux lettres : C-t
  • Inverser deux lignes : C-x t
  • Supprimer le caractère sous le curseur : C-d
  • Mettre le mot à partir du curseur (jusqu'au prochain espace) en majuscule : M-u
  • Mettre le mot à partir du curseur (jusqu'au prochain espace) en minuscule : M-l

Infos

  • Il existe plusieurs versions "légeres" d'emacs par exemple mg ou uemacs
