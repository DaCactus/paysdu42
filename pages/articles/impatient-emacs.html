Impatient Emacs

(^ = Touche Ctrl)

Débuter avec Emacs

  • Ouvrir emacs : Taper emacs -nw dans votre terminal.
  • Mettre un header : ^C ^H
  • Fermer emacs : ^X ^C
  • Ouvrir un fichier : emacs "chemin du fichier"
  • Ouvrir un fichier dans emacs déjà ouvert : ^X ^F
  • Changer le mode d'emacs (couleurs) : Alt X 'mode'-mode (exemple : C-mode, html-mode, ect)
  • Annuler ce que je suis en train de demander : ^G
  • Annuler (revenir en arrière) : ^ /
  • Ne pas utiliser en mode graphique : emacs -nw

Partager la fenêtre emacs

  • Séparer la fenêtre en 2 verticalement : ^X 3
  • Séparer la fenêtre en 2 horizontalement : ^X 2
  • Aller à l'autre fenêtre : ^X O
  • Supprimer une fenêtre : ^X 0

Fonctionnalités sympas

  • Faire une tabulation : Alt I
  • Completer automatiquement un mot deja ecrit : Alt /
  • Rechercher un mot : ^S
  • Remplacer un mot par un autre : Alt X 'replace-string'
  • GDB dans emacs : Alt X 'gdb'. A ouvrir avec le partage de fenêtre pour voir les étapes une par une dans le fichier source.
  • Parcourir un dossier : ^X D
  • Selectionner une zone : ^ espace puis les fleches haut et bas pour selectionner.
    Arreter la selection avec ^ G
  • Commenter la selection : ^ C C
  • Decommenter : ^ U C C

Infos

  • Il existe une version "legere" d'emacs : mg
